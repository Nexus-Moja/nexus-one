# Architecture

## High-Level Architecture

NEXUS-ONE is composed of a set of portal applications, shared platform services, and infrastructure components that work together to support scheduling, dispatch, billing, reporting, and compliance workflows.

## Core Layers

- Applications: patient, driver, dispatch, fleet, facility, billing, compliance, executive, and admin portals
- Platform Services: identity, authorization, notifications, messaging, payments, audit, reporting, maps, and AI
- Shared Services: UI libraries, reusable components, localization, accessibility, and utilities
- Infrastructure: deployment, observability, and environment management
