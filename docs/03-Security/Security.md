# Security

## HIPAA

The platform is designed with HIPAA-aligned controls, including secure handling of protected health information, access controls, and auditable workflows.

## RBAC

Role-based access control ensures users only access the functions and data required for their role.

## Authentication

Authentication is implemented through secure identity providers with multifactor support and strong password policies.

## Authorization

Authorization policies enforce least-privilege access across applications and platform services.
