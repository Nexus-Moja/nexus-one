# NEXUS ONE v1.0.4 Fix Notes

- Language selector now mounts inside an explicit navigation slot immediately after Book a Ride.
- React and every static portal page include the same navigation slot.
- MutationObserver re-homes the selector when React finishes rendering.
- Removed floating positioning through final CSS overrides.
- Stripe checkout URL is environment-configurable through `VITE_STRIPE_PAYMENT_URL`.
- Booking references are appended as Stripe `client_reference_id` for reconciliation.
- Stripe campaign parameters identify NEXUS ONE web bookings.

This release uses Stripe hosted checkout. Server-side webhook reconciliation still requires valid production Stripe server credentials configured only in the deployment environment.
