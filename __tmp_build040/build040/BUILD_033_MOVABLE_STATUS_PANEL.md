# Build 033 — Movable Livecare Status Panel

- The Moving, Patient, Available, and Attention filter panel can be dragged anywhere within the Livecare map.
- Dragging is constrained to the map boundaries, so the panel cannot be lost off-screen.
- The selected position is remembered in the browser.
- Double-click the Move handle to reset the panel to the right-center position.
- Status and service filtering remain functional.
